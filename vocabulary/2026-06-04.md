【 1. String 】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
• 音标: /strɪŋ/
• 词性: n. (名词)
• 释义: 字符串；一串字符 (编程里用来存文字的)
• 父子笔记:
  孩子笔记写了“字符串”。在代码里，凡是用引号包起来的字，比如 "Hello"，就是 String。
• 实用例句:
  - My name is a String.(我的名字是一串字符。)
  - String message = "Well done!";(代码写法：定义一个字符串变量叫message。)

【 2. Score 】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
• 音标: /skɔːr/
• 词性: n. (名词)
• 释义: 分数；得分；成绩
• 父子笔记:
  编程里经常用来记录游戏赢了得多少分，或者考试考了多少分。
• 实用例句:
  - The game score is 100.(游戏分数是100。)
  - High score (高分).

【 3. Print 】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
• 音标: /prɪnt/
• 词性: v. (动词)
• 释义: 打印；输出；印刷
• 关联词:
  ➡️ Printer (打印机)
• 父子笔记:
  编程第一步就是让屏幕显示出字，用的就是 print。
• 实用例句:
  - Print this word.(打印/输出这个词。)
  - System.out.print("Hello");(代码写法：在屏幕上打印Hello。)

【 4. Int 】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
• 音标: /ɪnt/ (Integer的缩写)
• 词性: 数据类型
• 释义: 整数类型 (专门用来存整数的)
• 父子笔记:
  孩子在这里打了个“？”，问是不是整数类型。Bingo！完全正确！它就是 Integer 的缩写。
• 实用例句:
  - int age = 10; (代码写法：定义一个整数变量age，值是10。)
  - int numberOfStudents = 30;

【 5. Scan / Scanner 】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
• 音标: /skæn/ /ˈskænər/
• 词性: v. (动词) / n. (名词)
• 释义: 
  - Scan: 扫描
  - Scanner: 扫描仪；扫描器
• 父子笔记:
  和 Print 对应，Print 是往外显示，Scan 是往里读取。编程里常用 Scanner 来接收键盘输入。
• 实用例句:
  - Use a scanner to read the code.(用扫描仪读码。)
  - Scanner input = new Scanner(System.in); (代码写法：准备一个扫描器来接收输入。)

【 6. Input 】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
• 音标: /ˈɪnpʊt/
• 词性: n./v. (名词/动词)
• 释义: 输入
• 记忆法:
  ➡️ 对应 Import (导入)。In 是“进”，把数据“进”到电脑里。
• 父子笔记:
  编程里指用户告诉电脑的信息（比如打字、点击）。
• 实用例句:
  - Keyboard input (键盘输入).
  - Get user input (获取用户输入).

【 7. Output 】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
• 音标: /ˈaʊtpʊt/
• 词性: n./v. (名词/动词)
• 释义: 输出
• 记忆法:
  ➡️ 对应 Export (导出)。Out 是“出”，把数据“出”给屏幕或文件。
• 父子笔记:
  和 Input 是一对反义词。Print 其实就是一种 Output。
• 实用例句:
  - Show the output on screen.(在屏幕上显示输出。)
  - Input and output (输入与输出，常缩写为 I/O).

【 8. Value 】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
• 音标: /ˈvæljuː/
• 词性: n. (名词)
• 释义: 价值；数值
• 父子笔记:
  孩子非常聪明地把 Value 和“取值”联系起来了！在编程里，Value 就是指变量里具体存着的那个数或那个字。
• 实用例句:
  - The value of X is 5.(X的值是5。)
  - Get the maximum value (获取最大值).

【 9. Import 】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
• 音标: /ˈɪmpɔːrt/
• 词性: v. (动词)
• 释义: 导入；进口
• 记忆法:
  ➡️ Im (向内) + Port (港口) 👉 货物进港！
• 父子笔记:
  编程里开头经常写 import，意思是“借来用一下别人的工具包”。
• 实用例句:
  - Import a library (导入一个库文件).
  - We import goods from other countries.(我们从国外进口货物。)

【 10. Export 】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
• 音标: /ˈekspɔːrt/
• 词性: v. (动词)
• 释义: 导出；出口
• 记忆法:
  ➡️ Ex (向外) + Port (港口) 👉 货物出港！
• 父子笔记:
  和 Import 相反，把做好的程序或文件给别人用。
• 实用例句:
  - Export the file (导出文件).
  - China exports many products.(中国出口很多产品。)
