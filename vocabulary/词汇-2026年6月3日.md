【 1. Mac / Macintosh 】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
• 音标: /mæk/ /ˈmækɪntɒʃ/
• 词性: n. (名词)
• 释义: 
  - Mac: 苹果电脑 (口语简称)
  - Macintosh: 麦金塔电脑 (苹果电脑的全称)
• 父子笔记: 
  孩子记录了缩写和全称的对应关系，这是苹果电脑的专属名称。
• 实用例句:
  - My brother has a new Mac.(我弟弟有一台新的Mac电脑。)
  - This software only works on Macintosh.(这个软件只能在苹果电脑上运行。)

【 2. OS / Operating System 】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
• 音标: /ˌoʊˈes/ /ˈɑːpəreɪtɪŋ ˈsɪstəm/
• 词性: n. (名词)
• 释义: 
  - OS: 操作系统的缩写 (Operating System)
  - Operating: 操作的 (动词 operate 的现在分词)
  - System: 系统
• 父子笔记: 
  孩子非常敏锐地把 OS 拆成了 Operation(操作) 和 System(系统)。虽然准确的英文是 Operating，但孩子的理解逻辑是完全正确的！
• 实用例句:
  - The OS helps us run apps.(操作系统帮我们运行软件。)
  - An operating system manages computer hardware.(操作系统管理着电脑硬件。)

【 3. Window vs Windows 】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
• 音标: /ˈwɪndoʊ/ vs /ˈwɪndoʊz/
• 词性: n. (名词)
• 释义: 
  - Window: 窗口 (单数，泛指任何一个窗口)
  - Windows: 视窗系统 (复数，首字母大写时特指微软公司的操作系统)
• 父子笔记: 
  区分大小写和单复数的关键！小写的 window 是屏幕上的方框，大写的 Windows 是那个熟悉的四色旗标志。
• 实用例句:
  - Close that window, please.(请把那个窗口关上。)
  - My dad's computer uses Windows.(我爸爸的电脑用的是Windows系统。)

【 4. Software / Soft / Ware 】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
• 音标: /ˈsɔːftwer/ /sɔːft/ /wer/
• 词性: n. (名词) / adj. (形容词) / n. (名词)
• 释义: 
  - Software: 软件 (计算机程序的总称)
  - Soft: 柔软的 (词根)
  - Ware: 器皿；物品 (词根，常作后缀)
• 父子笔记: 
  孩子的拆词逻辑非常棒！Soft(软的) + Ware(东西) = 软件。就像 Hardware(硬件) = Hard(硬的) + Ware(东西)。
• 实用例句:
  - We need to update the software.(我们需要更新软件。)
  - This pillow is soft.(这个枕头是柔软的。)

【 15. Learn / Learning 】

• 词性: v. (动词) / v.-ing (动名词)
• 释义: 
  - Learn: 学习 (动词原形)
  - Learning: 学习 (正在进行的状态；或者作为名词“学问”)

• 实用例句:
  - I want to learn Java.(我想学Java。——原形)
  - Learning Java is fun!(学习Java真有趣！——动名词做主语)
